## Установка

```bash
npm install
# or
yarn
```
## Присупайя к работе

Сначала запустите сервер разработки:

```bash
npm run dev
# or
yarn dev
```

Открыть [http://localhost:3000](http://localhost:3000) с помощью вашего браузера, чтобы увидеть результат.
